# Project Rules / 프로젝트 규칙

## Documentation Language / 문서 언어
- **Structure**: All documentation must be divided into two distinct sections.
- **Order**: The first section must be the **Full English Version**. The second section must be the **Full Korean Version**.
- **Separation**: Use a horizontal rule (`---`) to separate the two versions.

- **구조**: 모든 문서는 두 개의 명확한 섹션으로 나누어야 합니다.
- **순서**: 첫 번째 섹션은 **전체 영문 버전**이어야 하며, 두 번째 섹션은 **전체 국문 버전**이어야 합니다.
- **구분**: 두 버전 사이에는 수평선(`---`)을 사용하여 구분합니다.
